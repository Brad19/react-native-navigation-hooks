## `withNavigationProvider`

Create a higher-order React component class that passes the [NavigationContext](contexts#NavigationContext) values and into your component derived from supplied options.

```tsx
import { Navigation } from 'react-native-navigation'
import { withNavigationProvider, NavigationProvider } from 'react-native-navigation-hooks'

Navigation.registerComponent('MyScreenComponent', () => withNavigationProvider(MyScreenComponent))
```
