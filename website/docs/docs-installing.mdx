## Requirements

- node >= 8
- react-native >= 0.51
- react-native-navigation ">=2.21.0"

## npm or yarn

- `npm install --save react-native-navigation-hooks`

...or if you use `yarn`:

- `yarn add react-native-navigation-hooks`
